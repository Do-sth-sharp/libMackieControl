# Mackie Control 协议
## 什么是 Mackie Control 协议
Mackie Control 协议是由 Mackie 公司在 MIDI 协议的基础上设计的一套设备控制协议。该协议利用 MIDI 系统保留事件、音符开启/关闭事件、CC事件、弯音轮事件以及通道压力事件传递控制信息。
